# 🌱 Garden App

A professional command-line gardening assistant that provides seasonal advice, plant-specific tips, and answers gardening questions.

---

## 🚀 Features

- 🌍 Hemisphere-based seasonal advice
- 🌿 Plant-specific recommendations
- ❓ Question & Answer system
- 💾 Persistent user settings
- 📁 External JSON data storage
- 🧪 Basic unit testing

---

## 🛠 Tech Stack

- Python 3
- JSON (data storage)
- CLI interface

---

## ▶️ How to Run

```bash
python main.py
